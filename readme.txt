Provides a head-to-head against real players integration of a backgammon board into your wordpress website.

== Description ==

This plugin provides a way to connect your website to an interactive head-to-head backgammon board play management system at <a href="http://www.simplybg.com/play.html">http://www.simplybg.com/play.html</a>.  The integration is done via an Iframe that links to <a href="http://www.simplybg.com/play.html">http://www.simplybg.com/play.html</a>.  Players identify themselves via Google+ or Facebook connection or by specifically providing an email address themselves.  Future versions may support integrating user logins with Wordpress, the current version does not integrate user logins.

== Features ==

* Play head-to-head with other real players
* Player ELO rating system.
* Play single and multi-game Matches.
* Match-play lengths up to 21.
* Play with or without the Crawford rule.
* Play with or without the doubling cube.
* Compatible with desktop and mobile devices. Play anywhere, anytime.
* Integration via shortcode or widget technique.

== Installation ==

Dowload the plugin to your WP site.
Enabled the plugin.
